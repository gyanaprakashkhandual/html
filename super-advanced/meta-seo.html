Essential Meta Tags

<meta charset="UTF-8">

Character encoding. Always first. UTF-8 supports all languages.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Responsive scaling. Required for mobile. viewport-fit=cover for notched phones.

<meta name="description" content="150–160 char description">

Shown in search result snippets. Most important meta tag for CTR.

<meta name="robots" content="index, follow">

noindex, nofollow, noarchive, nosnippet, max-snippet:N, max-image-preview:large

<meta name="theme-color" content="#f43f5e">

Browser UI color in Android Chrome. Different value per color scheme via media attribute.

<meta name="color-scheme" content="dark light">

Declares supported color schemes — browser renders UI chrome accordingly.

Link Tags for SEO & PWA

<link rel="canonical" href="https://example.com/page">

Prevents duplicate content penalty — tells search engines which URL is the authoritative version

<link rel="alternate" hreflang="en" href="https://example.com/en/page">

International SEO — tells Google which language/region version to show

<link rel="manifest" href="/manifest.json">

PWA manifest — enables Add to Home Screen, offline support, splash screen

<link rel="icon" type="image/svg+xml" href="/favicon.svg">

SVG favicon scales perfectly to any resolution. Also add PNG fallback.

<link rel="apple-touch-icon" sizes="180x180" href="/apple-icon.png">

iOS home screen icon — 180×180px PNG, no transparency, rounded corners applied by OS

<link rel="sitemap" type="application/xml" href="/sitemap.xml">

Helps crawlers discover all pages

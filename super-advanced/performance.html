HTML Performance

Resource Hints

dns-prefetch
<link rel="dns-prefetch" href="//api.example.com">

Resolves DNS early for third-party origins — saves ~100–300ms

preconnect
<link rel="preconnect" href="https://fonts.googleapis.com">

DNS + TCP + TLS handshake early. Use for critical third-party origins.

preload
<link rel="preload" as="font" href="font.woff2" crossorigin>

High-priority fetch for critical resources used in current page.

prefetch
<link rel="prefetch" href="/next-page.js">

Low-priority fetch for likely next navigation. Cached for future use.

modulepreload
<link rel="modulepreload" href="/app.js">

Preloads ES modules including dependency graph parsing.

Script Loading Strategies

Normal
<script src="app.js"></script>
HTML → BLOCK → Resume

Blocks HTML parsing while fetching and executing. Avoid for non-critical scripts.

async
<script async src="app.js"></script>
Fetch parallel → execute ASAP

Fetches in parallel, executes as soon as downloaded (may interrupt parsing). Good for analytics.

defer
<script defer src="app.js"></script>
Fetch parallel → execute after DOM

Fetches in parallel, executes after DOM is ready. Preserves order. Best for most scripts.

Image Performance

Lazy Loading
<img loading="lazy" src="image.jpg" alt="...">

Browser-native lazy loading — don't load off-screen images until needed

fetchpriority
<img fetchpriority="high" src="hero.jpg" alt="..."> // LCP image

Hint browser to prioritize LCP image fetch — improves Core Web Vitals

srcset & sizes
<img
  srcset="img-320.jpg 320w, img-640.jpg 640w, img-1280.jpg 1280w"
  sizes="(max-width: 600px) 320px, (max-width: 1200px) 640px, 1280px"
  src="img-640.jpg" alt="Responsive">

Browser picks optimal image based on screen DPI and viewport

decoding="async"
<img decoding="async" src="image.jpg" alt="...">

Decode image off main thread — prevents jank during page load

width + height = No CLS
<img width="400" height="300" ...>

Browser reserves space before image loads — prevents Cumulative Layout Shift

format="avif/webp"
<source type="image/avif" srcset="img.avif">

AVIF = 50% smaller than JPEG. Use picture for format negotiation.

Content Security Policy & Meta Tags

<meta http-equiv="Content-Security-Policy"
  content="default-src 'self'; script-src 'self' 'nonce-ABC123'; style-src 'self' fonts.googleapis.com">

CSP prevents XSS — restrict which scripts and resources can load on your page

<meta name="referrer" content="no-referrer-when-downgrade">

Controls how much referrer info is sent in requests

<meta name="robots" content="index, follow, max-image-preview:large">

Controls crawler behavior — index, noindex, follow, nofollow

<meta name="theme-color" content="#f43f5e" media="(prefers-color-scheme: light)">

Browser UI color — shown in mobile browser chrome and PWA

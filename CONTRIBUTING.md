# Contributing to Learn HTML

Thank you for your interest in contributing! All skill levels are welcome.

---

## How to Contribute

1. **Fork** this repository
2. **Create a branch** — `git checkout -b feature/your-topic`
3. **Add or improve** HTML examples
4. **Commit** — `git commit -m "Add: example for topic"`
5. **Push** — `git push origin feature/your-topic`
6. **Open a Pull Request**

---

## Contribution Guidelines

- Place files in the correct folder (`basic/`, `intermediate/`, `advanced/`, `super-advanced/`)
- Use clear, descriptive filenames — e.g. `forms-validation.html`
- Write clean, well-commented HTML
- One topic per file
- Do not break existing examples

---

## Ideas for Contributions

- Fix typos or improve comments
- Add missing HTML features or tags
- Improve existing examples for clarity
- Add accessibility improvements

---

## Questions?

Open an [issue](https://github.com/gyanaprakashkhandual/html/issues) and we'll help you out.

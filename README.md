# Learn HTML

A structured, open-source repository for learning HTML — from the very basics to advanced techniques.

---

## Structure

```
html/
├── basic/            # Tags, headings, paragraphs, links, images, lists
├── intermediate/     # Forms, tables, media, semantic elements
├── advanced/         # APIs, canvas, SVG, accessibility, meta
└── super-advanced/   # Web components, performance, SEO, integrations
```

---

## Getting Started

```bash
git clone https://github.com/gyanaprakashkhandual/html.git
cd html
```

Open any `.html` file in your browser — no build tools required.

---

## Who Is This For?

| Level                 | Folder            |
| --------------------- | ----------------- |
| Complete beginners    | `basic/`          |
| Knows the basics      | `intermediate/`   |
| Comfortable with HTML | `advanced/`       |
| Ready to go deep      | `super-advanced/` |

# Security Policy

## Overview

This is a static HTML learning repository. It contains no backend, no user data, and no executable server-side code. The security risk is minimal by nature.

---

## Reporting a Vulnerability

If you discover a security concern (e.g. malicious code in a contribution, harmful external links, or scripts with unintended behavior), please **do not open a public issue**.

Instead, report it privately:

- **GitHub:** Open a private security advisory via the [Security tab](https://github.com/gyanaprakashkhandual/html/security)
- **Email:** Contact the maintainer directly through their [GitHub profile](https://github.com/GyanaprakashKhandual)

---

## What to Include in Your Report

- A clear description of the issue
- The affected file(s) and line numbers
- Steps to reproduce, if applicable

---

## Response

Reports will be acknowledged within **72 hours** and resolved as quickly as possible.
